# Kavi's Tours

A portfolio website for an actor, writer and coach.

### Link to project: https://niallam22.github.io/kavis-tours.github.io/
![Setting up a meeting](kavis-tours.gif?raw=true "Booking a meeting")

## About this project

This is a clean, modern and light weight design that is built to to take advantage of larger displays and seamlessly scaling down to fit all manner of mobile devices.

This project has been built with HTML, CSS, SCSS and Javascript.

Features:

- Single page light weight design
- Interactive carousel
- Mobile friendly
